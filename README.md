Solid Waste Collection Optimization in Ikitelli Organized Industrial ZoneOverview
This repository contains the source code and documentation for our graduation project, which focuses on optimizing industrial solid waste collection operations in the Istanbul Ikitelli Organized Industrial Zone (IOSB). By modeling the system as a Capacitated Vehicle Routing Problem (CVRP) with Split Delivery, we aim to minimize operational costs, redundant travel distances, and fuel consumption.  Key FeaturesOptimization Approach: Developed a Simulated Annealing (SA) meta-heuristic algorithm in Python to overcome the NP-Hard limitations of classical Mixed Integer Linear Programming (MIP) models.  Operational Efficiency: Implemented a "Split Delivery" operator to allow partial loading, enabling the fleet to handle high-volume waste sites that exceed single-vehicle capacities.  Sustainability: Designed as a green reverse logistics decision support system, directly contributing to UN Sustainable Development Goals (SDG 11, 12, and 13) by reducing fuel consumption and operational inefficiencies.  Performance: Achieved an 87.28% average capacity utilization rate across the fleet while maintaining a balanced and humane workload distribution for personnel.  Technical StackLanguage: Python  Algorithm: Simulated Annealing (SA) with dynamic penalty functions  Environment: Designed for complex combinatorial optimization problems with modular state decoder functions  Results SummaryTotal Waste Collected: 170,200 units (100% service coverage).  Optimized Distance: 203.68 kilometers.  Fleet Utilization: 8 active vehicles completing 15 optimized trips.  Time Management: All operations completed within the 480-minute legal shift limit.
